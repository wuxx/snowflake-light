# snowflake-light
Touch stepless dimming snowflake-light, open source hardware, TYPE-C interface power supply, stepless dimming through finger touch. It can be used as a pendant decoration, or to create a romantic atmosphere in the room, or as a handicraft to send to friends. More color versions will be launched later.
